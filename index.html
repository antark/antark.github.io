本小站

所谓“好记性不如烂笔杆”，本站为个人实现，就是为了记录自己的学习、工作和生活等等。组织很简单， 以 项目 为主要划分， 包含众多小 标签 。

之前的站点建立在 SAE 之上，使用的 drupal 框架， 这绝对是一个坑 (垃圾 SAE !)。后来自己使用 Node 实现了一个简单站点，就是看到的这个，最初尝试运行在 BAE 之上， 后来发现 BAE 也不很理想 ( 收费呀有木有 ! ) ，再后来就使用了 OpenShift 。

本站使用的是一些开源技术搭建而成，主要有 Node + Express + MongoDB + jQuery + Bootstrap + FontAwesome + 其他。
